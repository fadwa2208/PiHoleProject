# Ad-Blocking Using Pi-hole

## Description

Our project aims to use a Raspberry Pi as an ad-blocker. Using Debian as our distribution, we will install Pi-Hole (ad-blocking application) in order to block advertisements and to prevent compagnies to track our personal data on a network level. This will be done by connecting the Raspberry Pi to the modem of our home. Our goal is to protect users from unwanted content and to protect their network as a whole. ## Getting Started

### Dependencies

This project requires a Raspberry Pi model 3B+ with it's kit, a Linux Distribution, and a Pi-hole.

### Installing

* Install Pi-hole from: https://pi-hole.net/
* use following command to install Pi-hole: curl -sSL https://install.pi-hole.net | bash

### Set Up Tutorial

* Buy Raspberry Pi model 3B+ with it's kit.
* Install a Rapberry Pi operating system.
* Connect and configure the Raspberry Pi to the modem.
* Install Pi-hole.
* Set up desired Settings
